## 1. 이미지 워터마크 삽입 기능 도입 제안
안녕하세요, **Juno**입니다. AI가 생성한 창작물의 출처를 밝히고 브랜딩을 강화하기 위해, 저장 시점에 자동으로 워터마크를 삽입하는 기능을 제안합니다.

## 2. 도입 배경 및 효과
- **브랜딩**: 생성된 이미지가 공유될 때 전시회 정보를 자연스럽게 노출.
- **저작권 보호**: 무단 도용을 방지하고 원본 출처를 명확히 함.
- **전문성 강화**: 관람객에게 기념품으로서 더 높은 퀄리티의 결과물 제공.

## 3. 기술적 구현 방안
- **라이브러리**: `sharp` (High-performance Node.js image processing)
- **적용 시점**: `src/app/api/generate/route.ts` 내에서 이미지를 파일로 저장하기 직전에 실행.
- **작업 내용**:
    - AI 결과 이미지 Buffer 생성.
    - 워터마크(텍스트 또는 로고) Buffer 생성.
    - `composite()` 함수를 이용한 레이어 합성.

## 4. 디자인 레이아웃 제안
