# 260309_나스_배포보고서

본 문서는 **Arthur-App** 프로젝트의 UI 개선 사항을 로컬에서 완료하고, 이를 GitHub을 거쳐 나스(NAS) 테스트 서버에 반영하는 절차를 정리한 결과 보고서입니다.

## 1. 주요 수정 사항 (Summary)

### 1.1. 캐릭터 데코 요소 추가 및 최적화
- **캐릭터 배치**: `Deco_01.png`(좌측)와 `Deco_02.png`(우측, 구 Deco.png)를 카드 상단에 고정 배치.
- **파일명 변경**: 명확한 관리를 위해 `Deco.png`를 `Deco_02.png`로 파일명 일원화.
- **애니메이션 제어**: 사용자의 요청에 따라 부유(Floating) 애니메이션을 제거하고 정적 위치로 고정.

### 1.2. 모바일(아이폰 16 Pro) 환경 최적화
- **Hydration 오류 수정**: 서버와 클라이언트 간의 HTML 불일치로 발생하는 에러를 해결하기 위해 `isClient` 렌더링 가드 적용.
- **반응형 줄바꿈**: 하단 안내 문구에 `<wbr />` 태그를 적용하여 다양한 화면 크기에서 자연스럽게 줄바꿈이 일어나도록 조절.
- **결과 이미지 모달 최적화**: 
  - 모달 상하 패딩 축소 (`2.5rem` → `1.5rem`)
  - 로딩 상태와 완료 상태의 박스 크기를 동적으로 분리 (로딩창은 기존의 넉넉한 사이즈 유지)

## 2. 배포 절차 (NAS Deployment Guide)

### 2.1. 코드 동기화
GitHub에 푸시된 최신 코드를 나스 서버로 가져옵니다.
```bash
git pull origin main
```

### 2.2. 도커 이미지 빌드 및 재실행
이미지 파일명 변경 및 정적 파일 경로 수정을 반영하기 위해 빌드를 새로 진행합니다.
```bash
# 컨테이너 중지 및 새 빌드 실행
docker-compose up -d --build
```
