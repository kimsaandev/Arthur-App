## 1. AI 이미지 생성 서비스 비용 분석 보고서

## 2. 단위 생성 비용 예측 (Pricing Unit)
현재 Google Cloud Vertex AI 및 Gemini 모델의 표준 단가를 기준으로 산출했습니다.

### 2.1. API 호출 비용
- **Standard Image Generation**: $0.02 / image
- **Upscaling / Editing**: $0.01 / image
- **한화 환산**: 1회 생성당 **약 30원 ~ 50원** (환율 1,350원 기준)

### 2.2. 인프라 유지 비용
- **클라우드 스토리지 (GCS/S3)**: 월 1GB당 약 30원 (이미지 1장당 1MB 기준, 약 1,000장 저장 시 30원 미만으로 매우 저렴)
- **네트워크 트래픽**: 이미지 다운로드 및 화면 전송 시 발생 (장당 1원 미만)

## 4. 부하 관리 및 비용 최적화 방안

### 4.1. 서버 부하 분산 (Load Balancing)
- **대기열 관리**: 대량의 요청이 들어올 경우 API 할당량(Quota) 초과 오류가 발생할 수 있으므로, 서버단에서 초당 요청 수(QPS)를 조절하는 로직 적용.
- **Failover**: AI 모델 장애 시 미리 준비된 기본 이미지(Placeholder)를 보여주어 관람객의 경험 저해 최소화.

### 4.2. 불필요한 비용 제거
- **자동 삭제 정책 (TTL)**: 전시가 끝난 후 일정 기간이 지난 데이터는 자동으로 삭제하여 관리비용 최소화.
- **클라이언트 캐싱**: 동일 가이드 이미지나 UI 리소스는 브라우저 캐시를 사용하여 트래픽 절감.
