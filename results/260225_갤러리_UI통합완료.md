## 1. 개요
기존에 분리되어 있던 메인 갤러리(`/gallery`)와 인스타 갤러리(`/insta-gallery`)를 하나의 페이지로 성공적으로 통합했습니다.

## 2. 세부 작업 내역
이번 통합 작업을 통해 개선된 UI와 데이터베이스 변경 사항은 다음과 같습니다.

### 2.1. UI 통합 및 개선 (Gallery)
- **카드 컴포넌트 구조 변경**: 하나의 카드 안에 두 장의 이미지가 위아래로 나란히 표시되도록 구조를 변경했습니다.
    - **상단**: AI가 생성한 원본 이미지 전시 (`task.image`)
    - **하단**: 인스타 프레임이 적용된 두 번째 이미지 전시 (`task.instaImage`)
- **개별 다운로드 기능 탑재**: 각각의 이미지 영역 우측 하단에 독립적인 다운로드 버튼(`AI 원본 다운`, `인스타 다운`)을 배치하여 원하는 버전만 다운로드할 수 있도록 했습니다.
- 과거 데이터처럼 인스타 프레임 이미지가 없는 단일 구성의 경우 기존처럼 상단의 AI 원본 이미지만 보이도록 유연하게 대응했습니다.

### 2.2. 인스타 갤러리 폐기
- 기능이 통합됨에 따라 불필요해진 `/insta-gallery` 전용 페이지 코드는 프로젝트에서 완전히 삭제했습니다.
- 이를 통해 유지보수 포인트가 줄어들어 장기적인 관리가 훨씬 편해졌습니다.
