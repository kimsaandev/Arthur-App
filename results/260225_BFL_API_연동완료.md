## 1. 개요
앱 관리자(Admin) 페이지에서 신규로 도입한 `Flux.2` AI 모델의 API 통신 제공처를 임시 설정했던 Replicate에서 제작사인 **Black Forest Labs(BFL) 공식 API 규격**으로 전면 교체했습니다.

## 2. 주요 변경 사항

### 2.1. 어드민 UI 텍스트 업데이트 (`/admin/page.tsx`)
- 관리자가 혼동하지 않도록 UI 상의 명칭들을 BFL 공식 통신 규격에 맞춰 일괄 수정했습니다.
- "Flux.2 Klein 9B (Replicate)" -> **"Flux.2 Klein 9B (Black Forest Labs)"**
- "Replicate Token" -> **"BFL Token"**

### 2.2. 통신 백엔드 구조 개편 (`/api/generate/route.ts`)
- 기존에 Replicate 플랫폼에 요청을 보내던 통신 코드(fetch)를 걷어내고, BFL 공식 문서의 `image_to_image` 통신 규격으로 전면 재작성했습니다.
- **주요 수정 포인트**:
  - 엔드포인트: `api.replicate.com` -> `api.bfl.ml`
  - 인증 헤더: `Authorization: Token` -> BFL 전용인 `X-Key` 방식으로 전환
  - Base64 데이터 전처리: BFL 서버는 `data:image/jpeg;base64,` 같은 접두어(Prefix)를 받지 못하므로, 서버 전송 전 해당 글자들을 깨끗하게 잘라내는 정제 로직(`replace`)을 추가로 구현했습니다.
  - 결과 수신 메커니즘: BFL의 비동기 결과 수신 방식에 맞춰 작업(Task) 생성 후 실시간 추적(Polling)하는 알고리즘으로 동기화했습니다.
